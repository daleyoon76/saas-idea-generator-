# 독스헌트(DocShunt) 상세 분석

> 조사일: 2026-03-01

## 기본 정보

| 항목 | 내용 |
|------|------|
| **URL** | [docshunt.ai](https://docshunt.ai/) |
| **운영사** | 사페레아우데 |
| **슬로건** | "아이디어만 입력해도 전문가 수준 사업계획서 완성" |
| **생성 시간** | 약 3분 |
| **결과물 분량** | 평균 20~30장 워드 문서, 15종 콘텐츠 |

## 핵심 기능

### 1. AI 사업계획서 자동 생성

- 아이디어(아이템 정보)를 입력하면 AI가 사업계획서 초안 생성
- 자료나 이미지 첨부 가능 → 약 3분 후 결과물 완성

### 2. 포함 섹션 (확인된 것)

- 시장 분석 (TAM/SAM/SOM 포함, 도표 시각화)
- 고객 페르소나
- 린캔버스 (BM 구조화)
- 경쟁 분석 / 경쟁사 비교
- 마케팅 전략
- 사업성 분석
- 레퍼런스/출처 (AI 검색 RAG 기반)

### 3. 지원사업 양식 자동 채우기 (Auto-Fill & Export)

- 정부 지원사업 기관별 양식 업로드 → AI가 내용을 자동으로 채워줌
- 동일 내용을 양식별로 재작성하는 비효율 제거
- 400+개 지원사업 양식 대응 가능

### 4. AI 챗봇 에이전트 수정

- 워크스페이스 내에서 AI 챗봇으로 내용 수정/보완 가능

### 5. 레퍼런스 탐색

- AI가 구글 검색으로 신뢰성 있는 데이터를 별도 확보
- 결과물에 반영 + 요약/링크 형태로 제공

| 플랜 | 가격 (정가) | 비고 |
|------|-----------|------|
| **체험판** | 무료 | 기능 제한 |

- **월 구독이 아닌 1회 이용권 방식으로 추정** (확인 필요)

## 타겟 고객

- 정부 지원사업 신청자 (예비창업패키지, 초기창업패키지 등)
- 예비 창업자 / 처음 사업계획서 쓰는 사람
- 사업계획서에 리소스를 많이 할애하는 스타트업 대표/실무자

## My CSO 대비 비교

| 비교 축 | 독스헌트 | My CSO |
|---------|---------|--------|
| **주 타겟** | 정부 지원사업 신청자 | 스타트업/신규사업 기획자 |
| **AI 구조** | 단일 LLM 호출 (추정) | 5개 전문 에이전트 순차 협업 |
| **검색 소스** | Google 검색 (RAG) 단일 | Tavily + Reddit + Google Trends + Product Hunt 4-way |
| **자기 검증** | 없음 | Devil's Advocate 내장 (섹션 14) |
| **양식 대응** | 400+개 지원사업 양식 자동 채우기 | 자체 14개 섹션 구조 (양식 커스텀 없음) |
| **수정 방식** | AI 챗봇으로 인라인 수정 | 현재 없음 (생성 후 .md/.docx 다운로드) |
| **가격 모델** | 1회 이용권 (89,000~149,000원) | 미정 (현재 오픈소스) |
| **할루시네이션 억제** | RAG 기반 출처 제공 | 출처 강제 + "가상 수치 금지" 시스템 규칙 |
| **기존 기획서 활용** | 양식에 내용 채우기 | Import → 에이전트 팀이 심화/업그레이드 |

## 독스헌트에서 배울 점

1. **지원사업 양식 Auto-Fill** — 한국 시장에서 킬러 기능. My CSO도 향후 지원사업 양식 대응 고려 가치 있음
2. **AI 챗봇 인라인 수정** — 생성 후 수정 UX가 편리. My CSO는 현재 다운로드만 가능
3. **가격 모델** — 1회 이용권 방식이 한국 B2C에서 진입 장벽 낮춤
